# A-Soul 直播信息推送bot  
### A-Soul livestream infomation monitor bot for Telegram  
## Feature:  

 本项目旨在实时获取和提炼A-Soul成员直播相关信息并推送至Telegram机器人，如：  
  + 成员开播提醒  
  + 包括弹幕数量、观看人数的实时同接信息
  + 自动筛选推送重复次数较高的弹幕信息
  + SC信息
  + ~~待定~~
